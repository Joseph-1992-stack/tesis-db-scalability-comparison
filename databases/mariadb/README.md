# MariaDB Dataset Parameters

## Descripción

Actualmente esta carpeta no contiene archivos activos.

La parametrización de datasets utilizada por MariaDB se encuentra centralizada en:

`databases/postgres/loader/tpcc_params.ps1`

Este archivo es compartido por ambos gestores para garantizar equivalencia metodológica durante los experimentos.

## Observación

La existencia de esta carpeta responde a una decisión de organización del repositorio y facilita una futura migración hacia una estructura común para ambos gestores.
